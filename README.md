## ✨ Features

* 📈 **Live Market Tracking:** Fetches real-time prices for National and International Stocks, ETFs, and Mutual Funds using the Yahoo Finance API.
* 🪙 **Precious Metals Tracker:** Live value tracking for Gold and Silver investments.
* 💵 **Cash Management:** Keep a record of your liquid cash or bank balances.
* 📉 **Profit/Loss Analytics:** Automatically calculates current value, absolute profit/loss, and percentage returns for each asset class.
* 🏦 **Liability & Debt Tracker:** Dedicated section to log your active loans/debts, which are dynamically deducted from your total assets.
* 📊 **Real-time Net Worth Calculation:** A central dashboard that sums up all assets, subtracts liabilities, and displays your real-time Net Worth.

---

## 🛠️ Tech Stack

* **Frontend:** Pure HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Backend:** Python (Flask)
* **API Integration:** Yahoo Finance API (`yfinance` library)
* **Database:** SQLite
